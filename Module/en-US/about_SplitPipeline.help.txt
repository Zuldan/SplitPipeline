SHORT DESCRIPTION
    SplitPipeline - Parallel Data Processing in PowerShell

LONG DESCRIPTION
    The only cmdlet is Split-Pipeline. It splits the input, processes parts by
    parallel pipelines, and outputs data for further processing. It works
    without collecting the entire input, large or infinite.

    Get help:
    PS> Import-Module SplitPipeline
    PS> help -Full Split-Pipeline
